# Texture Assets

This folder contains texture images for the 3D building modeler.

## Structure

```
textures/
├── walls/
│   ├── brick.jpg
│   ├── concrete.jpg
│   ├── glass-curtain.jpg
│   ├── stucco.jpg
│   └── wood-panel.jpg
├── roofs/
│   ├── shingle.jpg
│   ├── metal.jpg
│   ├── tile.jpg
│   └── green-roof.jpg
└── ground/
    ├── grass.jpg
    └── asphalt.jpg
```

## Adding Textures

To add textures to this project:

1. Place JPG or PNG texture images in the appropriate subdirectory
2. Ensure textures are tileable for best results
3. Recommended resolution: 512x512 to 2048x2048 pixels
4. Keep file sizes reasonable (< 1MB per texture)

## Free Texture Resources

You can find free tileable textures at:
- [Poly Haven](https://polyhaven.com/textures)
- [Textures.com](https://www.textures.com/)
- [CC0 Textures](https://cc0textures.com/)
- [3D Textures](https://3dtextures.me/)

## Placeholder Textures

For now, the application will work with colored materials if textures are not loaded.
You can add your own textures by uploading them through the UI or by placing files in this directory structure.
